# Changelog

## [0.1.0] - 2025-03-23

### Added
- `cet explain` — explain any code file in plain English
- `cet pr` — review staged git changes like a senior engineer  
- `cet spec` — generate OpenAPI 3.1 specs from route code
- `cet cache` — manage local response cache
- `.cet.toml` project config with team conventions support
- GitHub Actions example for automated PR reviews
- 48 tests covering all tools, prompts, chunker, and config
